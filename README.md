# Npm Run! [![NPM version][npm-image]][npm-url]

> nr, An alias for npm run

## Getting Started

### Install:

```
npm install -g npmrun
```

### How to use:

```
nr script_name
```

### Examples:

```
nr test
```

![nr test](docs/images/nr_test.gif)


```
nr start
```

![nr start](docs/images/nr_start.gif)
